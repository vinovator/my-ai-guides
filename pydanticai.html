Philosophy

Built by the creators of Pydantic. It brings production-grade engineering to Agents: Type Safety, Dependency Injection, and Streaming by default. It is not a graph, it's a framework for reliable function execution.

1. Basic Agent

from pydantic_ai import Agent

# Define agent with a system prompt
agent = Agent(
    'openai:gpt-4o',
    system_prompt='Be concise and helpful.'
)

# Run synchronously
result = agent.run_sync('Where does "hello world" come from?')
print(result.data)

2. Dependency Injection

Unlike other frameworks using global variables, PydanticAI injects dependencies (DB connections, API keys) into tools at runtime.

from dataclasses import dataclass
from pydantic_ai import RunContext

@dataclass
class MyDeps:
    user_id: int
    db_conn: str

agent = Agent('openai:gpt-4o', deps_type=MyDeps)

@agent.tool
async def get_user_name(ctx: RunContext[MyDeps]) -> str:
    # Access deps safely
    print(f"Fetching for {ctx.deps.user_id} using {ctx.deps.db_conn}")
    return "Alice"

result = agent.run_sync('Who am I?', deps=MyDeps(user_id=123, db_conn="postgres://..."))
