Smolagents (Hugging Face)

Philosophy

Most agents output JSON. Smolagents (formerly transformers.agents) output Code. The LLM writes a Python script to solve the task, and the framework executes it in a sandbox. This is often more efficient than multiple JSON turn-taking.

CodeAgent (The Core)

from smolagents import CodeAgent, HfApiModel

# 1. Define Model (e.g., Qwen, Llama, or OpenAI)
model = HfApiModel(model_id="Qwen/Qwen2.5-Coder-32B-Instruct")

# 2. Create Agent
agent = CodeAgent(
    tools=[], 
    model=model, 
    add_base_tools=True # Adds search, calculator, etc.
)

# 3. Run
# The agent will write python code to solve this, print the result, and return it.
result = agent.run("What is the 10th fibonacci number multiplied by the distance to the moon in km?")
print(result)
