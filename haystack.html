Pipelines Components Code Example

Haystack (Deepset)

Philosophy

Haystack (v2.0+) is built on Pipelines. You explicitly connect components (Retrievers, Generators, Routers) together. It is explicit, debuggable, and modular.

Basic RAG Pipeline

from haystack import Pipeline
from haystack.components.retrievers.in_memory import InMemoryBM25Retriever
from haystack.components.generators import OpenAIGenerator
from haystack.components.builders import PromptBuilder

# 1. Initialize Components
retriever = InMemoryBM25Retriever(document_store=doc_store)
generator = OpenAIGenerator(model="gpt-4")

template = """
Answer the question based on context: {{ documents }}
Question: {{ question }}
"""
prompt_builder = PromptBuilder(template=template)

# 2. Build Pipeline
pipe = Pipeline()
pipe.add_component("retriever", retriever)
pipe.add_component("prompt_builder", prompt_builder)
pipe.add_component("llm", generator)

# 3. Connect them
pipe.connect("retriever", "prompt_builder.documents")
pipe.connect("prompt_builder", "llm")

# 4. Run
pipe.run({"retriever": {"query": "What is X?"}, "prompt_builder": {"question": "What is X?"}})
