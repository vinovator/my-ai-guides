Pipelines Components RAG Example

Haystack (Deepset)

Explicit, modular pipelines for RAG and Agents.

1. Philosophy

Haystack (v2) uses a directed graph of Components. You explicitly connect `output` of one to `input` of another. It's great for debugging complex flows.

Technique: Building a Pipeline

from haystack import Pipeline
from haystack.components.retrievers.in_memory import InMemoryBM25Retriever
from haystack.components.generators import OpenAIGenerator
from haystack.components.builders import PromptBuilder

# 1. Define Components
prompt_template = """
Given the context: {{ documents }}
Answer: {{ query }}
"""
builder = PromptBuilder(template=prompt_template)
llm = OpenAIGenerator(model="gpt-4")

# 2. Add to Pipeline
pipe = Pipeline()
pipe.add_component("retriever", retriever)
pipe.add_component("builder", builder)
pipe.add_component("llm", llm)

# 3. Connect (Explicit Wiring)
pipe.connect("retriever.documents", "builder.documents")
pipe.connect("builder", "llm")

# 4. Run
pipe.run({"retriever": {"query": "Hello"}, "builder": {"query": "Hello"}})
