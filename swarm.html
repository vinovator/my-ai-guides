The Handoff Pattern

Swarm (experimental) is built on one simple idea: Handoffs. An agent is just a set of instructions and tools. One of those tools can be "Transfer to Agent B".

Unlike CrewAI or AutoGen, Swarm has no global state management or complex graph. It's just function calls returning a new Agent object.

Basic Implementation

from swarm import Swarm, Agent

client = Swarm()

# 1. Define Handoff Function
def transfer_to_spanish_agent():
    """Transfer spanish speaking users to this agent."""
    return spanish_agent

# 2. Define Agents
english_agent = Agent(
    name="English Agent",
    instructions="You only speak English.",
    functions=[transfer_to_spanish_agent]
)

spanish_agent = Agent(
    name="Spanish Agent",
    instructions="Tu solo hablas espa√±ol.",
)

# 3. Run
response = client.run(
    agent=english_agent,
    messages=[{"role": "user", "content": "Hola. I want to speak spanish."}]
)

print(response.messages[-1]["content"])

Context Variables

Pass data between agents using context variables.

def greet(context_variables):
    name = context_variables.get("name", "User")
    return f"Hello, {name}!"

agent = Agent(functions=[greet])

response = client.run(
    agent=agent,
    messages=[{"role": "user", "content": "Hi!"}],
    context_variables={"name": "Alice"}
)
