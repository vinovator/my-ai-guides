Philosophy

Phidata focuses on solving the Memory and Knowledge problem. It treats Agents as "Assistants" that have built-in persistent storage (Postgres) and knowledge bases (Vector DBs).

1. Assistant with Storage

Automatically saves chat history to a database.

from phi.assistant import Assistant
from phi.storage.agent.postgres import PgAgentStorage

storage = PgAgentStorage(table_name="agent_sessions", db_url="postgresql://...")

agent = Assistant(
    name="MemoryAgent",
    storage=storage,
    show_tool_calls=True,
    add_history_to_messages=True, # Remembers context
)

agent.print_response("My name is Bob.")
# Restart script...
agent.print_response("What is my name?") # Returns "Bob"

2. Assistant with Knowledge

Connects to vector databases (PDFs, Wikis) effortlessly.

from phi.knowledge.pdf import PDFUrlKnowledgeBase
from phi.vectordb.pgvector import PgVector2

knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://example.com/report.pdf"],
    vector_db=PgVector2(collection="reports", db_url="...")
)
# Load once
knowledge_base.load(recreate=False)

agent = Assistant(knowledge_base=knowledge_base, search_knowledge=True)
agent.print_response("Summarize the report.")
